# Minesweeper
This is a 5x5 Minesweeper Android game developed on Android Studio for the Mobile Software Development class in AIT Budapest Fall 2016.

Features: 
- Flagging tiles
- Revealing tiles
- Randomized mine generation
- Surrounding mine counter
- Restarting game
- Winner Detection
